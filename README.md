# CatTyper

Hey there! This is a simple typing test I made because I wanted something clean and smooth without all the clutter. Just open it and start typing. That's literally it.

## What it does

* **Looks nice** – Clean black and white design, nothing flashy
* **Feels smooth** – Words fade in nicely, cursor blinks, everything moves like butter
* **Dark mode** – For when you're typing at 2am and don't want to blind yourself
* **Replay button** – Watch how you typed after you finish (kinda satisfying ngl)
* **Pick your style** – Do 10 words, 30 words, 50 words, or go by time (5s to 60s or whatever you want)

## How to use it

1. Open the file in your browser
2. Start typing. Any key works. Don't overthink it
3. Press Tab to start over
4. Click the Dark button if you want dark mode

No install. No signup. No "create an account" nonsense.

## The buttons

Just type normally. Spacebar goes to next word. Backspace fixes mistakes (yeah you can go back to previous words too). Tab restarts everything.

## What the numbers mean

* **WPM** – How fast you type (the main number everyone cares about)
* **Raw** – Your speed if you ignored mistakes
* **Accuracy** – Percentage of letters you got right
* **Consistency** – How steady your rhythm was (basically did you type like a robot or a caffeinated squirrel)
